Instrucciones de Configuración

Copia y pega estos comandos en tu terminal para aplicar los cambios en el sistema:

# 1. Crear carpetas necesarias
mkdir -p /home/usuario/.iconos

# 2. Limpiar animaciones de Lubuntu (el turpial dando vueltas)
sudo rm /usr/share/plymouth/themes/lubuntu-logo/spinner/animation-*.png
sudo rm /usr/share/plymouth/themes/lubuntu-logo/spinner/throbber-*.png
sudo rm /usr/share/plymouth/themes/lubuntu-logo/spinner/spinner.plymouth

# 3. Aplicar tu logo de carga (asumiendo que está en tu carpeta .iconos)
sudo cp /home/usuario/.iconos/spinner.plymouth /usr/share/plymouth/themes/lubuntu-logo/spinner/spinner.plymouth
sudo cp /home/usuario/.iconos/arranque.png /usr/share/plymouth/themes/lubuntu-logo/spinner/animation-0001.png
sudo cp /home/usuario/.iconos/arranque.png /usr/share/plymouth/themes/lubuntu-logo/spinner/throbber-0001.png

# 4. Cambiar el icono del menú de inicio
sudo cp /home/usuario/.iconos/lxqt-about.svg /usr/share/icons/hicolor/scalable/apps/lxqt-about.svg
sudo cp /home/usuario/.iconos/lxqt-about.svg /usr/share/icons/hicolor/scalable/apps/lxqt.svg
sudo cp /home/usuario/.iconos/mainmenu.svg /usr/share/lxqt/themes/ambiance/mainmenu.svg
sudo cp /home/usuario/.iconos/wall.png /usr/share/sddm/themes/lubuntu/wall.png

# 5. Aplicar cambios al sistema de arranque
sudo update-initramfs -u -k all

# 6. Para cambiar los iconos de office
sed -i 's|^Icon=.*|Icon=/home/usuario/iconos/excel|' ~/Desktop/Microsoft\ Office\ Excel\ 2003.desktop
sed -i 's|^Icon=.*|Icon=/home/usuario/iconos/word|' ~/Desktop/Microsoft\ Office\ Word\ 2003.desktop
sed -i 's|^Icon=.*|Icon=/home/usuario/iconos/powerpoint|' ~/Desktop/Microsoft\ Office\ PowerPoint\ 2003.desktop

echo "¡Configuraciones cargadas con éxito!"

# 7. Para instalar office
sudo apt install wine
mkdir -p ~/disco_office
sudo mount -o loop ~/Descargas/"OFFICE 11 03 SPA.iso" ~/disco_office
cd ~/disco_office
wine SETUP.EXE
